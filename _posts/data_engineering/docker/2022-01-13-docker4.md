### Docker - MongoDB

```bash
# mongoDB container run using named volume
docker run --name mongodb1 -e MONGO_INITDB_ROOT_USERNAME=root -e MONGO_INITDB_ROOT_PASSWORD=1234 -v mongo_data:/data/db -d -p 27017:27017 mongo:5.0.5
```
